## Requirements
### Requirement: 脚本规则编辑区选中反馈可见
系统 MUST 在 Tauri 桌面端脚本规则编辑区中显示文本选中的可视高亮。

#### Scenario: 选择脚本内容
- **当** 用户在脚本规则编辑区拖拽选择文本
- **那么** 选中区域应有可见高亮并与浅色主题一致

## Requirements
### Requirement: 缩略图开关标签保持单行
桌面端工具栏中的“缩略图”开关标签 MUST 保持单行显示，不得因窗口变窄而自动换行。

#### Scenario: 缩窄窗口宽度
- **当** 用户将窗口宽度调整到较窄
- **那么** 工具栏“缩略图”开关标签保持单行显示（不换行）

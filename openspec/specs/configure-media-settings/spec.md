## 需求
### 需求：桌面端媒体设置入口
桌面端 MUST 在文件列表工具栏提供“设置”入口，点击后打开媒体设置弹窗。

#### 场景：打开设置弹窗
- **当** 用户点击工具栏“设置”
- **那么** 弹出媒体设置弹窗

### 需求：媒体设置集中展示
桌面端媒体设置弹窗 MUST 提供图片查看器、视频播放器、缩略图开关三项设置，并在主界面工具栏不再单独展示这些控件。

#### 场景：查看设置弹窗
- **当** 用户打开媒体设置弹窗
- **那么** 弹窗中可配置图片查看器、视频播放器、缩略图开关
- **并且** 工具栏不再显示图片查看器/视频播放器按钮与缩略图开关

### 需求：设置作用域保持现状
桌面端 MUST 保持现有设置作用域：图片查看器与视频播放器为全局设置，缩略图开关为配置级设置。

#### 场景：切换配置
- **当** 用户切换不同配置
- **那么** 图片查看器与视频播放器设置保持不变
- **并且** 缩略图开关状态跟随配置存储值

### 需求：设置变更轻提示
桌面端在媒体设置变更后 MUST 显示“设置已保存”的轻提示。

#### 场景：修改媒体设置
- **当** 用户选择或清除图片查看器/视频播放器，或切换缩略图开关
- **那么** 弹出“设置已保存”的轻提示

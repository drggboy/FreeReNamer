## ADDED Requirements
### Requirement: 支持选择图片查看器与视频播放器
系统 MUST 允许用户分别选择图片查看器与视频播放器应用。

#### Scenario: 选择媒体查看器
- **当** 用户为图片或视频选择自定义查看器
- **那么** 系统应显示当前选择并提供清除入口

### Requirement: 打开媒体文件优先使用自定义应用
系统 MUST 在打开媒体文件时优先使用用户配置的查看器。

#### Scenario: 打开图片文件
- **当** 用户打开图片文件且已配置图片查看器
- **那么** 系统应调用配置的应用打开
- **并且** 若配置不可用应回退到系统默认应用

### Requirement: Web 端查看器选择受限
系统 MUST 在 Web 环境禁用媒体查看器选择功能。

#### Scenario: Web 端查看器按钮
- **当** 用户在 Web 端查看媒体查看器选择按钮
- **那么** 按钮应显示禁用状态

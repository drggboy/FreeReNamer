# 变更：桌面端媒体设置收拢到设置弹窗

## 为什么
主界面同时展示图片查看器、视频播放器、缩略图三个按钮/开关，界面占用过高且操作分散，需要收拢到统一的设置入口。

## 变更内容
- 桌面端文件列表工具栏保留一个“设置”按钮（位置与当前按钮区域一致）
- 点击“设置”按钮弹出媒体设置弹窗，集中配置图片查看器、视频播放器与缩略图开关
- 维持现有设置作用域：图片查看器与视频播放器为全局设置，缩略图开关为配置级设置
- Web 端保持现状（设置按钮仍为禁用提示）

## 影响
- 受影响规范：file-list-header、configure-media-settings（新增）
- 受影响代码：src/components/file/files-panel.tauri.tsx（工具栏与设置入口）、可能新增设置弹窗组件
